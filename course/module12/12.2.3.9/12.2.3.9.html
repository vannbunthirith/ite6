The figure on this page is an interactive activity that checks the learner’s understanding of data protection terminology. Term: Degaussing, Description: Uses powerful magnets to eliminate the magnetic field on a hard drive. Term: Trusted Platform Module, Description: Must be enabled in BIOS before using BitLocker. Term: Principle of Least Privilege, Description: Users are limited to only the resources they need on the computer or network. Term: Security Key Fob, Description: A small device used for two-factor authentication. Term: Software Firewall, Description: A program that runs on a computer to allow or deny traffic between the computer and other computers to which it is connected.
