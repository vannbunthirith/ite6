Wireless Security Modes

Use a wireless encryption system to encode the information being sent to prevent unwanted capture and use of data. Most wireless access points support several different security modes, as shown in the figure. As discussed in a previous chapter, always implement the strongest security mode (WPA2) possible.

Many routers offer WiFi Protected Setup (WPS). WPS allows very easy WiFi security setup. With WPS, both the router and the wireless device will have a button that, when both are pressed, automatically configures WiFi security between the devices. A software solution using a PIN is also common. It is important to know that WPS is not secure. It is vulnerable to brute-force attack. WPS should be turned off as a security best practice.

The figure on this page is an image of the Wireless-N broadband router utility that shows the Wireless tab with the Security Mode dropdown menu expanded that lists the following modes: Disabled, WEP, WPA Personal, WPA Enterprise, WPA2 Personal, and WPA2 Enterprise. WPA2 Personal is selected.
