Web Security

There are various web tools (e.g., ActiveX, Flash) that can be used by attackers to install a program on a computer.

To prevent this, browsers have features that can be used to increase web security:

  • ActiveX Filtering

  • Pop-up Blocker

  • SmartScreen Filter

  • InPrivate Browsing

The figure on this page is a diagram that shows a large circle labelled as, ‘Browser Security’. Around the large circle and overlapping the large circle are four smaller circles. The smaller circles are labelled as, ‘ActiveX Filtering’, ‘Pop-up Blocker’, ‘SmartScreen Filter’, and ‘InPrivate Browsing’.
