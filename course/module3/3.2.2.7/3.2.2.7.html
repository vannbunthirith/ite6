Activity – Identify Terms

.

The figure on this page is an interactive activity that checks the learner’s understanding of the following terms and their descriptions. Term: CMOS, Description: Stores the BIOS settings. Term: UEFI, Description: Provides a mouse-enabled software interface instead of the traditional BIOS windows. Term: BIOS, Description: A small program that reads the configured settings stored in CMOS to determine how to configure the hardware. Term: LoJack, Description: A security feature that provides device information and location. Term: POST, Description: A hardware check on the main components of the computer. Term: TPM, Description: A chip designed to secure hardware by storing encryption keys, digital certificates, passwords, and data.
