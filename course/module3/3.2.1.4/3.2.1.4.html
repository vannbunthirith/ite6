UEFI Setup Program

Some computers run Unified Extensible Firmware Interface (UEFI). UEFI configures the same settings as traditional BIOS but also provides additional options. For example, UEFI provides a mouse-enabled software interface instead of the traditional BIOS screens.

Note: The UEFI setup screens in this section are for reference only and most likely will not look the same as yours. Please consider them as a guide and refer to your motherboard manufacturer documents.
