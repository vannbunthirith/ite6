Activity – Matching Email Protocols

.

The figure on this page is an interactive activity that checks the learner’s understanding of email protocols. SSL, developed to transmit files securely. All data exchanged between the email client and the email server is encrypted. IMAP, allows the user to organize email on the network email server, and to download copies of email. The original email remains on the email server. Default port is 143. Exchange, an email server, contact manager, and calendaring software created by Microsoft. POP3, retrieves emails from a remote server over TCP/IP. Normally does not leave a copy of email on the server. Default port is 110. SMTP, a text-based protocol that transmits emails across a TCP/IP network. Default port 25. MME, extends the email format to include text in ASCII standard as well as other formats, such as pictures and word processor documents.
