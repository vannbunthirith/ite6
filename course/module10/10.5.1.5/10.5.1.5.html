Verify Full System Functionality and Implement Preventive Measures

After you have corrected the problem, verify full functionality and, if applicable, implement preventive measures. The figure shows a list of the steps to verify the solution.

The figure on this page is a table that lists step 5 in troubleshooting Mobile, Linux, and OS X problems, verify full system functionality and implement preventive measures. Verify solution and full system functionality for mobile device operating systems: Reboot the mobile device, browse the Internet using Wi-Fi, browse the internet using 4G, 3G, or another carrier network type, make a phone call, send a text message, open different types of apps. Verify solution and full system functionality for Linux and OS X: Reboot the computer, browse the Internet using Wi-Fi, browse the Internet using a wired connection, send a test email, open different programs.
