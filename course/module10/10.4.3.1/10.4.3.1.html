File and Folder Commands

A number of command line tools are included in Unix-like systems by default. To adjust the command line tool operation, users can enter parameters and switches along with the command. See the figure for a few basic CLI commands

It is easy to create and edit text files in the CLI. The command vi opens a text editor. The command q is used to exit the editor when you are finished.

The figure on this page is a table that lists the following basic CLI commands. Command: man, Description: Displays the documentation for a specific command. Command: ls, Description: Displays the files inside the directory. Command: cd, Description: Changes the current directory. Command: mkdir, Description: Creates a directory under the current directory. Command: cp, Description: Copies files from source to destination. Command: mv, Description: Moves files to a different directory. Command: rm, Description: Removes files. Command: grep, Description: Searches for specific strings of characters within a file or other commandsâ€™ outputs. Command: cat, Description: Lists the contents of a file and expects the file name as the parameter.
