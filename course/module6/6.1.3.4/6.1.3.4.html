System Configuration

System Configuration (MSCONFIG) is a tool used to identify problems that keep Windows from starting correctly. To help with isolating the issue, services and startup programs can be turned off, and turned back on one at a time. After you have determined the cause, permanently remove or disable the program or service, or reinstall it.

The figure briefly explains the tabs and options available in System Configuration.

To open System Configuration, use the following path:

Control Panel > Administrative Tools > System Configuration

The figure on this page is a table that lists the following tabs and options available in System Configuration. Tab: General, Description: Displays the following list of startup selections; Normal startup – Starts normally, Diagnostic startup – Starts with basic services and drivers only, Selective startup - Starts with basic services and drivers and user-selected startup programs. Tab: Boot, Description: Displays a list of available Windows operating systems and shows boot options. Tab: Services, Description: Displays a list of the services that are started with the operating system, allows you to disable services for troubleshooting purposes. Tab: Startup, Description: Displays a list of all the applications that run when the computer boots up, this list includes the applications with manufacturer name, the executable file location, and the registry location. Tab: Tools, Description: Displays a list of diagnostic tools that you can launch.
