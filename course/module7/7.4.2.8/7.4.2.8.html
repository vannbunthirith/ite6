Activity – Protocol Definitions and Ports

The figure on this page is an interactive activity that checks the learner’s understanding of the following protocols and default ports. Protocol: Hypertext Transfer Protocol (HTTP), Default Port: 80. Protocol: Domain Name Service (DNS), Default Port: 53. Protocol: File Transfer Protocol (FTP), Default Port: 21. Protocol: Apple Filing Protocol (AFP), Default Ports: 548 or 427. Protocol: Post Office Protocol V3 (POP3), Default Port: 110. Protocol: Secure Shell (SSH), Default Port: 22. Protocol: Simple Mail Transfer Protocol (SMTP), Default Port: 25. Protocol: Internet Message Access Protocol (IMAP), Default Port: 143. Protocol: Hypertext Transfer Protocol Secure (HTTPS), Default Port: 443. Protocol: Server Message Block (SMB), Default Port: 445.
